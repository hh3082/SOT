# Gauss Curvature & Monge–Ampère Visualization

Interactive visualization accompanying lecture notes on **Mikami, *Stochastic Optimal Transportation*, §3.2** — the marginal problem for stochastic processes.

Shows how the Monge–Ampère equation encodes the Gauss curvature problem as an optimal transport map: the gradient $Du$ of a convex potential pushes the modified Cauchy reference distribution onto a prescribed target density.

## Controls

- **Target density peaks** — switch between 1, 2, or 3-mode target densities
- **Convexity scale** — adjust the curvature of the convex potential

## Hosting

Deployed via GitHub Pages. The `index.html` is fully self-contained (no dependencies) and supports light/dark mode.
